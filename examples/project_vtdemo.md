# VT-Demo 项目概述

## 项目简介
VT-Demo 是一个基于向量检索技术的智能问答系统，专注于能源行业知识库问答场景。

## 技术栈
- Python 3.11
- LangChain / LangGraph
- SQLite（向量存储）
- FastAPI
- React + TypeScript

## 核心功能
1. 文档上传与解析（PDF、Word、TXT）
2. 文本向量化与存储
3. 基于语义检索的问答
4. 多轮对话支持

## 核心指标
- 文档处理速度：50页/分钟
- 检索召回率：~85%
- 问答响应时间：< 2秒

## 技术架构
```
用户输入 → 向量检索 → LLM 生成答案 → 返回结果
```

## 团队情况
- 开发人数：2人
- 开发周期：3个月

## 风险点
1. 当前使用 pickle 存储向量，切换到 SQLite 后稳定性待验证
2. 依赖包体积较大（requirements.txt 超过 100MB）
3. 尚未做生产级优化（缓存、限流、监控）
